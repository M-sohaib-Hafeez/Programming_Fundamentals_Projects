# 💻 Programming Fundamentals Projects

A collection of C++ console-based projects built as part of the **Programming Fundamentals** course at **Dawood University of Engineering and Technology (DUET)**.

---

## 📁 Projects

| Project | File | Description |
|---------|------|-------------|
| 🏦 Banking System | `Banking_System.cpp` | Account creation, deposit, withdrawal, and balance management |
| 📦 Inventory Management System | `Inventory_Management_System.cpp` | Add, update, and track product inventory |
| 📚 Library Management System | `Library_Management_System.cpp` | Book issuing, returning, and record management |
| 🛒 Mini Online Store | `Mini_Online_Store.cpp` | Product listing, cart, and order simulation |
| 📝 Online Examination System | `Online_Examination_System.cpp` | MCQ-based exam with scoring and result display |
| ❓ Quiz System | `Quiz_System.cpp` | Interactive quiz with timer and scoring |
| 🎓 Student Management System | `Student_Management_System.cpp` | Student records: add, search, update, delete |
| ✅ To Do List | `To_Do_List.cpp` | Task management with add, complete, and delete features |

---

## ⚙️ How to Run

### Prerequisites
- A C++ compiler (GCC / G++ recommended)
- Any IDE: Code::Blocks, Dev-C++, VS Code with C++ extension
- For Quiz_System also download Question.txt file (or you can create your own but follow the same pattern )

### Compile & Run (Command Line)
```bash
g++ projects/Banking_System.cpp -o banking
./banking
```

Replace `Banking_System.cpp` with whichever project you want to run.
